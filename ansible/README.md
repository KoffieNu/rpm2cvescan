rscan_standalone
=========

Install and configures rscan tool.

Requirements
------------

Minimum OS requirement: CentOS 6.

Role Variables
--------------

There are variables that are in defaults/main.yml. Other than this you can use group_vars/servers.yml or host_vars/server-node.yml.
